# Circuit Breaker Panel Map

## Summary

Map of which breaker controls which room or circuit. Update this when you add or identify circuits.

## Details

Panel is in the garage, right wall.

| Breaker | Label | Controls |
|---------|-------|----------|
| 1 | Main | All |
| 3 | Kitchen | Kitchen outlets + refrigerator |
| 5 | Dishwasher | Dishwasher only |
| 7 | Microwave | Microwave only |
| 9 | Living Room | Living room outlets |
| 11 | Master Bedroom | Master bedroom outlets |
| 13 | Bedroom 2 | Bedroom 2 + hallway outlets |
| 15 | Bathrooms | Both bathrooms |
| 17 | HVAC | Furnace + thermostat |
| 19 | Garage | Garage outlets + opener |
| 21 | Exterior | Porch lights + exterior outlets |
| 23 | Dryer | Dryer (240V) |
| 25 | Washer | Washer only |

### Notes

- Breaker 13 sometimes trips when the space heater in bedroom 2 is running alongside the laptop charger. Consider moving one to a different circuit.
- The outdoor outlet by the back door is on breaker 21 but wasn't labeled — confirmed by testing.
